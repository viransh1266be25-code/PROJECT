# FitZone Hub

My first web development project! I built this during my university course to learn HTML, CSS, and JavaScript.

## Why I Built This

I wanted to create something useful that could actually help people. Since not everyone can afford a gym membership, I thought why not make a website with free workout videos? That way anyone with internet access can workout from home.

## What's Inside

Started with just the basic structure in index.html, then added styling in style.css. The JavaScript in script.js handles all the interactive stuff - filtering videos, calculating BMI, generating workout plans.

### Features I Added:
- Video gallery with thumbnails (took me a while to figure out how to get YouTube thumbnails working)
- Category filters - users can click buttons to see only chest, back, legs, or abs videos
- BMI calculator - simple math, weight divided by height squared
- Workout plan generator - three levels: beginner, intermediate, advanced

## Running the Project

Just double-click index.html to open it in your browser. No servers or installation required.

## Problems I Faced

Getting the video thumbnails to display properly was tricky. Had to look up how YouTube thumbnail URLs work. Also had to make sure the filter buttons update correctly when clicked.

The BMI calculator was straightforward but I had to handle the case where users enter nothing or invalid numbers.

## What I Would Add Next

More videos for sure. Maybe user accounts so people can save their progress. A search function would be nice too.
